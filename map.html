KAPSARC WebGIS

Web Map Viewer

Web Map Viewer is part of KAPSARC’s GIS portal, which allows you to view, create, save, and share interactive web maps.
Maps can be created in a few basic steps and opened in standard web browsers, mobile devices, and desktop map viewers.
They can be shared through links, embedded in websites, and used to create map-based web apps.

Anonymous users can view and add content to maps using publicly shared layers, but can’t save or share maps,
whereas Signing in to Web Map Viewer with an ArcGIS account allows you to get additional options for adding layers,
perform analysis, save your map to your personal workspace, and share the map with others.
